get_battery()​

Syntax​

get_battery()

Parameters​

None

Discription​

This function returns the current battery level percentage of the drone.

Returns​

The current battery percentage of the drone's battery.

Example Code​

#Python code
from codrone_edu.drone import *

drone = Drone()
drone.pair()
battery = drone.get_battery()
print(battery)
drone.close()
